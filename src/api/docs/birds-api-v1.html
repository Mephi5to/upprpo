    Птицы

    Добавить список новых птиц

    POST /api/v1/birds
    Body

    Media type: application/json

    Properties:

    • items: required (array of BirdBuildInfo)

      Данные о птицах

      • name: required (string)

        Название птицы

      • description: required (string)

        Описание птицы

      • imageFileId: required (string)

        Идентификатор файла с изображением птицы

      • audioFileId: required (string)

        Идентификатор файла с записью голоса птицы

    Response 200

    Body

    Media type: application/json

    Properties:

    • birds: required (array of Bird)

      Птицы

      • id: required (string)

        Идентификатор птицы

      • name: required (string)

        Название птицы

      • description: required (string)

        Описание птицы

      • imageFileId: required (string)

        Идентификатор файла с изображением птицы

      • audioFileId: required (string)

        Идентификатор файла с записью голоса птицы

    Response 400

    Некорретные данные запроса

    Запросить список птиц

    GET /api/v1/birds
    Query Parameters
    • offset: (number - default: 0 - minimum: 0)

      Позиция, начиная с которой нужно вернуть птиц

    • limit: (number - default: 100 - minimum: 0 - maximum: 1000)

      Максимальное количество птиц, которое нужно вернуть

    • name: (string)

      Название птицы

    Response 200

    Body

    Media type: application/json

    Properties:

    • birds: required (array of Bird)

      Птицы

      • id: required (string)

        Идентификатор птицы

      • name: required (string)

        Название птицы

      • description: required (string)

        Описание птицы

      • imageFileId: required (string)

        Идентификатор файла с изображением птицы

      • audioFileId: required (string)

        Идентификатор файла с записью голоса птицы

    Response 400

    Некорректные данные запроса

    Запросить информацию о ресурсе

    Response 200

    Headers
    • Allow: required (string)

      Example:

      GET,POST,OPTIONS

    URI Parameters

      Описание птицы

    • imageFileId: required (string)

      Идентификатор файла с изображением птицы

      Идентификатор файла с записью голоса птицы

    Response 404
